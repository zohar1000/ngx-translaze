TranslazeService is a service which handles all language text management, it is provided in root.

You can access the service properties/methods to get an information about the current language, change language, get the config object and so on.
An example of using the service in a component, the onClickChangeLang method is calling TranslazeService to change the current language.

Below is a list of the properties/methods:

properties:
Namedescription
text$ a BehaviorSubject for the current language text.
the value has the interface of the language text
lang$ a ReplaySubject for the current language, the value has the interface of TranslazeLang
isText$ a BehaviorSubject for indication if the language text is available or not, the value is boolean

methods:
Namedescriptionparametersreturn
changeLang change the current language langCode- the code that is defined for the language in the config
getConfig get the config object that was passed to TranslazeModule in AppModule config object of interface TranslazeConfig
translate replacing tokens in string with values.
this method is useful for cases where values are known only at run time, we can then prepare strings in our language text that contains placeholders and have them replaced during run time.

for example: translate('{{'{'}}name} is {{'{'}}age} years old', {{'{'}} name: 'John', age: 35 })
will return 'John is 35 years old'
str- string containing tokens where each token is wrapped with {{'{'}}}
values- object containing token/value pairs 		string where tokens are replaced with values
getInfo get information about the current language and the config object object having the interface of TranslazeInfo:
{{'{'}}
  config: the config object that was passed to TranslazeModule
  langCode: the language code from the config object
  langName: the language name from the config object
