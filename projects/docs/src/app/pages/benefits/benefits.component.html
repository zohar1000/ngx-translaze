Benefits of using dynamically loaded translation components
Working with the typescript compiler provides us with convenient and reliable tools right out of the box.
We can define an interface for our translation object and automatically have the following benefits:
  • The language text in the language components have the correct structure according to the language text interface
  • All references in our code and templates to the language text are safe and points to existing keys
  • Intellisense in our code and templates
  • Smaller bundle size since Angular's builtin tools are used
  • Language components are lazy loaded so they will be loaded only the first time they are required
  • Flexibility when coding the language text, it is a javascript object and not JSON so you can do things like:
